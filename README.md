# A ps1 and batch script to clean up your pc

# WARNING
## YOU DECIDE WHAT TO DELETE SO MAKE SURE YOU TAKE A GOOD LOOK ON EACH FILE BEFORE DECIDING TO DELETE
## ALL THE FILES THAT ARE AUTOMATICALLY DELETED ARE SAFE TO DELETE FILES THAT DO NOT AFFECT YOUR SYSTEM
## RUNNING RANDOM SCRIPTS YOU FOUND FROM THE INTERNET IS NEVER A GOOD IDEA SO MAKE SURE TO BE ON THE LOOKOUT


# How to Use:
- Download both the PCCleaner.ps1 and Run_PCCleaner.bat and place both in the same folder.
- then run the Run_PCCleaner.bat

## Standard
- Automatically cleans up common junk Windows leaves behind:
Temporary files your PC created and never cleaned up
Old Windows update files that are no longer needed
Cached files from browsers like Chrome, Edge, and Firefox
Whatevers sitting in your Recycle Bin
Leftover Windows component files that are safe to remove

## Deep Scan
- Looks for files that are may be useless but needs your approval(a window pops up for deciding on each file):
Large files that haven't been touched in a long time
Crash logs and error reports Windows quietly saves in the background
Old installer files sitting in your Downloads or Desktop
Leftover junk from apps that were uninstalled but didn't clean up after themselves

## App Reviewer
- Goes through your installed programs and finds ones you probably don't use anymore:
Checks the last time each app was actually used
Only flags apps that haven't been touched in 30+ days
Skips anything important like drivers, system tools, and Windows components
Lets you go through them one by one and mark what to uninstall(same proccess as the deep scan)
Launches each uninstaller automatically for anything you mark

## Speed Boost
- A collection of tweaks that make Windows run a bit leaner and faster:
Checks your startup programs and flags any suspicious or unnecessary ones (you still remove them manually via Task Manager)
Turns off Windows visual animations that slow things down
Switches your power plan to High Performance so your CPU isn't holding back
Clears your DNS cache, which can speed up website loading
Flushes RAM that Windows is holding onto but not actually using
Turns off Xbox Game Bar, which runs in the background even if you never use it
Runs a Windows system file check to find and flag any corrupted files

## Full Nuke
- Runs everything stated above automatically
